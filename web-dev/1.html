Einführung in HTML

Was ist HTML?

HTML oder Hyper Text Markup Language ist das Skelett-Framework des Internets. HTML ist extrem einfach zu erlernen und skaliert in der Komplexität der komplexeren Anforderungen, die wir haben. Genau wie Apex hat HTML für alles eine eigene Syntax, aber im Gegensatz zu Apex gibt es keine if-else-Anweisungen, es gibt keine Schleifen, es gibt keine "Logische" Operation innerhalb von HTML. HTML ist, vereinfacht ausgedrückt, eine Möglichkeit, ein Dokument zu formatieren, indem ihm Attribute hinzugefügt werden.

A HTML file ends with the extension .html, and uses tags within < >and </ > to identify how to display the content between the tags.

HTML schreiben

First, we need to define the .html file to be a HTML5 file. For this, we start off using <html>. In your text editor or IDE of choice (I'll be using Atom), create a new file and name it index.html and add the following content:

<html>
</html>

A thing to remember is, when you upload your site to a server, the server will ALWAYS look for a file named index.extension where extension could be .html, .php, .asp, etc. This is why all the first pages in every folder we create is called index.html. I spent an entire day when I was learning HTML back in 4th grade figuring out why the extremely expensive server won't display my page at harshdeephura.com but harshdeephura.com/about.html worked just fine.

Jetzt, da wir den Anfang und das Ende unserer Datei definiert haben! Jetzt teilen wir die HTML-Datei in zwei Abschnitte auf:

  • Leiter
  • Gremium

Leiter

Der Kopfbereich des HTML enthält alle Preloader wie CSS, SEO-Tags, Titel der Webseite und Analyseskripte (wir sprechen im JavaScript-Kurs über Skripte). Fügen wir unserer HTML-Datei einen Kopf hinzu und geben wir ihr einen Titel.

<html>
	<head>
		<title>Harshdeep's Portfolio</title>
	</head>
</html>

Save and open this in your browser. You'll notice the title says Harshdeep's Portfolio(or the text you chose to write)!

Gremium

Der Hauptteil wird nach dem Head-Tag geladen und enthält den gesamten Inhalt, den der Benutzer sehen wird. Um Text in unseren Textkörper zu schreiben, verwenden wir das thep-Tag, das für Absatz steht. Während Sie Text technisch direkt in das Body-Tag schreiben können und der Text immer noch angezeigt wird, wird es Probleme verursachen, wenn wir unseren Text wie das Hinzufügen von Zeilenumbrüchen und Attributen formatieren. Fügen wir ein Body-Tag hinzu und schreiben wir unseren Namen!

<html>
	<head>
		<title>Harshdeep's Portfolio</title>
	</head>
	<body>
		<p>Harshdeep Singh Hura</p>
	</body>
</html>

Großartig! Um jetzt einen neuen Absatz hinzuzufügen, können wir einfach mehr p-Tags erstellen! Schreiben wir darüber, wie viele Salesforce-Zertifizierungen wir wie jedes coole Kind auf LinkedIn haben!

<html>
	<head>
		<title>Harshdeep's Portfolio</title>
	</head>
	<body>
		<p>Harshdeep Singh Hura</p>
		<p>1x Salesforce Certified</p>
	</body>
</html>

Schreiben wir jetzt eine Zeile über uns selbst. Vielleicht kommen wir her, was wir tun, wenn wir Freizeit haben, wo wir arbeiten, alles! Oder Sie können "Interessen, die im Lebenslauf gut aussehen" googlen, weil wir ehrlich sein können, kein normaler Mensch kann sich an seine Interessen erinnern, wenn er ausdrücklich darum gebeten wird. Aber dieses Mal stellen Sie sicher, dass Sie mehrere Leerzeichen zwischen Ihren Linien hinzufügen. Sehen Sie sich diesen Code zum Beispiel an:

<html>
	<head>
		<title>Harshdeep's Portfolio</title>
	</head>
	<body>
		<p>Harshdeep Singh Hura</p>
		<p>1x Salesforce Certified</p>

		<p>I build iOS apps.
		I also binge TV Shows on Netflix instead of being a functioning member of the society.
		I also really REALLY love dogs!</p>
	</body>
</html>

Notice how all the lines were converted to a single line of text? This is because HTML doesn't care about how you format your text as long as it is between a start and end tag. To add a new line, we use a br tag called as break. Unlike most tags, br doesn't have a closing tag.

<html>
	<head>
		<title>Harshdeep's Portfolio</title>
	</head>
	<body>
		<p>Harshdeep Singh Hura</p>
		<p>1x Salesforce Certified</p>

		<p>I build iOS apps.<br>
		I also binge TV Shows on Netflix instead of being a functioning member of the society.<br />
		I also really REALLY love dogs!</p>
	</body>
</html>

Notice how I wrote br first as <br> and then as <br /> but they yield the same result? Because these tags don't require a separate closing tag, you can choose to write it either ways and HTML still accepts it like a loving mother.

Text formatieren

Jetzt möchte ich wirklich über meine Liebe zu Hunden sprechen. Die Großbuchstaben werden WIRKLICH nicht genug gerecht, also werden wir dem Text etwas Kursiv, Unterstreichung und Fettdruck hinzufügen!

Formatierung Tag
BOLD <strong>...</strong>
Kursiv <em>...</em>
Unterstreichen <u>...</u>
Streik <strike>...</strike>

Fügen wir dies jetzt unserem Code hinzu!

<html>
	<head>
		<title>Harshdeep's Portfolio</title>
	</head>
	<body>
		<p>Harshdeep Singh Hura</p>
		<p>1x Salesforce Certified</p>

		<p>I build iOS apps.<br>
		I also binge TV Shows on Netflix instead of being a functioning member of the society.<br />
		I also really <strong>REALLY</strong> love dogs!</p>
	</body>
</html>

Hmm. Ich denke immer noch, dass es nicht genug gerecht wird. Fügen wir weitere Formatierungen hinzu!

<html>
	<head>
		<title>Harshdeep's Portfolio</title>
	</head>
	<body>
		<p>Harshdeep Singh Hura</p>
		<p>1x Salesforce Certified</p>

		<p>I build iOS apps.<br>
		I also binge TV Shows on Netflix instead of being a functioning member of the society.<br />
		I also really <strong><u><em>REALLY</em></u></strong> love dogs!</p>
	</body>
</html>

There we go! Notice how to add multiple formatting options the tags are nested. It's a good practice to close the inner tags first and then the outer tags. We can further manipulate it, by underlining the whole REALLY love dogs! sentence.

<html>
	<head>
		<title>Harshdeep's Portfolio</title>
	</head>
	<body>
		<p>Harshdeep Singh Hura</p>
		<p>1x Salesforce Certified</p>

		<p>I build iOS apps.<br>
		I also binge TV Shows on Netflix instead of being a functioning member of the society.<br />
		I also really <strong><u><em>REALLY</em></strong> love dogs!</u></p>
	</body>
</html>

Um Nesting-Tags besser zu verstehen, können wir sie mit der Herstellung von Instant-Kaffee vergleichen. Es spielt keine Rolle, in welcher Reihenfolge Sie Ihren Instantkaffee, Zucker, Wasser und Milch aufgeben, es wird immer noch genau denselben Instantkaffee machen, aber gleichzeitig müssen Ihre Inhalte in der richtigen Reihenfolge geschrieben werden, so wie Getreide zuerst geht und dann die Milch.

Überschriften

Während ich Hunde wirklich liebe, möchte ich nicht, dass meine Liebe zu Hunden die Show stiehlt! Lassen Sie uns etwas mehr Wert auf den Namen legen. Ich möchte, dass mein Name eine größere Schriftgröße hat. Dafür werden wir das Überschriften-Tag verwenden.

Das Überschriften-Tag wird mit `h` geschrieben, gefolgt von einer Zahl von 1 - 6, 1 als die größte und 6 die kleinste. Lassen Sie uns unseren Code aktualisieren.

<html>
	<head>
		<title>Harshdeep's Portfolio</title>
	</head>
	<body>
		<h1>Harshdeep Singh Hura</h1>
		<p>1x Salesforce Certified</p>

		<p>I build iOS apps.<br>
		I also binge TV Shows on Netflix instead of being a functioning member of the society.<br />
		I also really <strong><u><em>REALLY</em></strong> love dogs!</u></p>
	</body>
</html>

Beachten Sie, wie ich das `p`-Tag durch `h1` ersetzt habe. Es ist nicht notwendig, `p`-Tags um ein `h#` herum zu haben, da sie immer in einer neuen Zeile beginnen.

Zusammenfassung

  • The html tag defines the start and end of our file. This tells the browser to render everything inside it using the HTML language.
  • The head tag is where our pre-loaders go, like our CSS and SEO tags
    • Das title-Tag definiert den Titel dieser Webseite.
  • The body tag is now loaded
    • The h1 tag defines a heading
    • Das p-Tag definiert einen neuen Absatz
    • The br tells the browser to start a new line
    • The strong is to bold the text
    • The u is to underline the text
    • The em is to italicize the text
      • Wir verschachteln Tags, um Effekte zu kombinieren
      • Das u-Tag wird erweitert, um mehr Text zu unterstreichen
