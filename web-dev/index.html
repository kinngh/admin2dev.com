HTML y CSS

¡Bienvenido al primer curso en Web Dev Track! Esta es una colección gratuita de tutoriales, videos, pruebas y más, todo diseñado para ayudarle a aprender los conceptos básicos del desarrollo web de manera efectiva.

Este curso repasa los fundamentos de HTML y CSS. Al final de este curso, podrá construir un sitio web de cartera, comprar un dominio y alojarlo en Netlify, ¡usando GitHub!

He tratado de acelerar este curso para tomar menos de una hora cada día en completarlo, por lo que incluso si te quedas atascado podrás encontrar una solución y apresurarte a aprender más rápido no dará mejores resultados. ¡Tómate tu tiempo!

Siempre puedes tuitearme @kinngh o unirte a nuestro servidor Discord para obtener ayuda.

Este es un curso continuo y se actualizará todos los días con un nuevo capítulo, y los capítulos anteriores se pueden actualizar en función de los comentarios de la comunidad.
