Mini curso de Git y colaboración

¡Bienvenido al Mini Curso de Git y Colaboración! Esta es una colección gratuita de tutoriales, videos, pruebas y más diseñados para ayudarte a aprender a usar Git de manera efectiva.

Este mini curso está escrito para desarrolladores de administradores de SF que han completado el curso Apex Starter y el mini curso VSCode

Siempre puedes tuitearme @kinngh o unirte a nuestro servidor Discord para obtener ayuda.

¡Partes de este curso fueron escritas en vivo en nuestro canal de YouTube! ¡Debería suscribirse para obtener más contenido de Salesforce y transmisiones en vivo!
