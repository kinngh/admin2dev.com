Git and Collaboration Mini Course

Welcome to Git and Collaboration Mini Course! This is a free collection of tutorials, videos, tests and more all designed to help you learn using Git effectively.

This mini course is written for SF Admins Developers who have completed the Apex Starter Course and VSCode Mini Course

You can always tweet me @kinngh or join our Discord Server for help.

Parts of this course was written live on our YouTube Channel! You should subscribe for more Salesforce Content and Live Streams!
