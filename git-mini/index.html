Git und Collaboration Minikurs

Willkommen beim Git and Collaboration Mini Course! Dies ist eine kostenlose Sammlung von Tutorials, Videos, Tests und mehr, die Ihnen helfen sollen, Git effektiv zu lernen.

Dieser Minikurs ist für SF Admins-Entwickler geschrieben, die den Apex Starter Kurs und den VSCode Mini Kurs abgeschlossen haben.

Sie können mich jederzeit @kinngh twittern oder unserem Discord-Server beitreten, um Hilfe zu erhalten.

Teile dieses Kurses wurden live auf unserem YouTube-Kanal geschrieben! Sie sollten sich für weitere Salesforce-Inhalte und Live-Streams anmelden!
