Mini-cours Git et collaboration

Bienvenue au mini-cours Git et Collaboration ! Il s'agit d'une collection gratuite de tutoriels, de vidéos, de tests et plus encore conçus pour vous aider à apprendre à utiliser Git efficacement.

Ce mini-cours est écrit pour les développeurs d'administrateurs SF qui ont terminé le cours Apex Starter et le mini-cours VSCode

Vous pouvez toujours me Twitter @kinngh ou rejoindre notre serveur Discord pour obtenir de l'aide.

Certaines parties de ce cours ont été écrites en direct sur notre chaîne YouTube ! Vous devriez vous abonner pour plus de contenu Salesforce et de diffusions en direct !

Outils
