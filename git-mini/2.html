Utilisation du terminal / CLI

L'utilisation de Terminal / Invite de commandes peut sembler super intimidante au début, mais une fenêtre de terminal n'est qu'une version texte de l'utilisation d'outils pointer-cliquer. Pensez-y de cette façon, chaque fois que vous créez un nouveau dossier, il exécute des commandes de terminal en arrière-plan. L'utilisation directe du terminal signifie que nous abandonnons les outils point et clic et que nous devenons des utilisateurs puissants de notre propre système.

On macOS Go to Applications > Utilities > Terminal or use Spotlight search to open Terminal. On Windows, press Command and R together, and type cmd.

Les deux outils sont des terminaux, sauf qu'ils ont des noms différents mais servent le même but. Apprenons quelques commandes de base.

CD

CD signifie Change Directory. Par défaut, lorsque vous ouvrez une nouvelle fenêtre de terminal, elle s'ouvre à la racine de l'utilisateur. Pour accéder au bureau, dans votre type de terminal (sensible à la casse) :

macOS :cd Desktop
Windows :cd C:\Users\(username)\Desktop

An alternative way to open Desktop folder in Windows is to type cd in terminal followed by a space and then dragging a folder in the terminal and removing the folder name before pressing enter.

Pour confirmer le fait que vous êtes maintenant dans le dossier Bureau, dans votre type de terminal :

macOS :ls
Windows :dir

Cela vous montrera une liste de tous les fichiers et dossiers disponibles dans le répertoire dans lequel vous vous trouvez !

Pour monter d'un niveau dans la hiérarchie des dossiers, vous utilisez :

macOS :cd ..
Windows :cd ..

This will change your directory to the folder that your current folder was in. So if you're in Desktop > Git folder, using cd.. will change your directory to Desktop.

MKDIR

MKDIR signifie Make Directory. Ceci est utilisé pour créer de nouveaux dossiers à votre chemin actuel. Alors que vous êtes encore dans Desktop, faisons un nouveau dossier appelé git. Assurez-vous qu'il s'agit d'un petit étui pour qu'il soit plus facile à taper.

macOS :mkdir git
Windows :mkdir git

Now in your desktop, a new folder called git has appeared!

CLEAR / CLS

La fenêtre du terminal peut devenir super désordonnée très rapidement ! Pour effacer notre fenêtre actuelle :

macOS :clear
Windows :cls

Cela efface le texte de la fenêtre du terminal et commence par un nouvel écran.
