Uso de Terminal / CLI

El uso de Terminal / Símbolo del sistema puede parecer súper intimidante al principio, pero una ventana de terminal es solo una versión de texto del uso de herramientas de apuntar y hacer clic. Piénsalo de esta manera, cada vez que creas una nueva carpeta, está ejecutando comandos de terminal en segundo plano. El uso directo del terminal significa que estamos abandonando herramientas de punto y clic y convirtiéndonos en usuarios avanzados de nuestro propio sistema.

On macOS Go to Applications > Utilities > Terminal or use Spotlight search to open Terminal. On Windows, press Command and R together, and type cmd.

Ambas herramientas son terminales, excepto que tienen nombres diferentes pero sirven para el mismo propósito. Aprendamos algunos comandos básicos.

CD

CD significa Change Directory. De forma predeterminada, cuando abre una nueva ventana de terminal, se abre en la raíz del usuario. Para llegar al escritorio, en su tipo de terminal (se distingue entre mayúsculas y minúsculas):

macOS:cd Desktop
Ventanas:cd C:\Users\(username)\Desktop

An alternative way to open Desktop folder in Windows is to type cd in terminal followed by a space and then dragging a folder in the terminal and removing the folder name before pressing enter.

Para confirmar el hecho de que ahora está en la carpeta Escritorio, en su terminal escriba:

macOS:ls
Ventanas:dir

¡Esto le mostrará una lista de todos los archivos y carpetas disponibles dentro del directorio en el que se encuentra!

Para subir un nivel en la jerarquía de carpetas, utilice:

macOS:cd ..
Ventanas:cd ..

This will change your directory to the folder that your current folder was in. So if you're in Desktop > Git folder, using cd.. will change your directory to Desktop.

MKDIR

MKDIR significa Make Directory. Esto se utiliza para crear nuevas carpetas en su ruta actual. Mientras todavía estamos en Desktop, hagamos una nueva carpeta llamada git. Asegúrese de que todo sea un estuche pequeño para que sea más fácil de escribir.

macOS:mkdir git
Ventanas:mkdir git

¡Ahora en tu escritorio, ha aparecido una nueva carpeta llamada git!

LIMPRAR / CLS

¡La ventana del terminal puede volverse súper desordenada muy rápido! Para borrar nuestra ventana actual:

macOS:clear
Ventanas:cls

Esto borra el texto de la ventana del terminal y comienza con una pantalla de aspecto fresco.
