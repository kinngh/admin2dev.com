Verwenden von Terminal / CLI

Die Verwendung von Terminal / Eingabeaufforderung mag auf den ersten Blick super einschüchternd aussehen, aber ein Terminalfenster ist nur eine Textversion der Verwendung von Point-and-Click-Tools. Stellen Sie es sich so vor, jedes Mal, wenn Sie einen neuen Ordner erstellen, werden Terminalbefehle im Hintergrund ausgeführt. Die direkte Nutzung des Terminals bedeutet, dass wir Punkt- und Klickwerkzeuge ablassen und Power-User unseres eigenen Systems werden.

On macOS Go to Applications > Utilities > Terminal or use Spotlight search to open Terminal. On Windows, press Command and R together, and type cmd.

Beide Werkzeuge sind Terminals, außer dass sie unterschiedliche Namen haben, aber demselben Zweck dienen. Lernen wir einige grundlegende Befehle kennen.

CD

CD steht für Change Directory. Wenn Sie ein neues Terminalfenster öffnen, wird es standardmäßig im Stammverzeichnis des Benutzers geöffnet. Um zum Desktop zu gelangen, in Ihrem Terminaltyp (Groß- und Kleinschreibung beachten):

macOS:cd Desktop
Fenster:cd C:\Users\(username)\Desktop

An alternative way to open Desktop folder in Windows is to type cd in terminal followed by a space and then dragging a folder in the terminal and removing the folder name before pressing enter.

So bestätigen Sie, dass Sie sich jetzt im Desktop-Ordner in Ihrem Terminaltyp befinden:

macOS:ls
Fenster:dir

Dadurch wird Ihnen eine Liste aller verfügbaren Dateien und Ordner in dem Verzeichnis angezeigt, in dem Sie sich befinden!

Um eine Ebene in der Ordnerhierarchie nach oben zu gehen, verwenden Sie:

macOS:cd ..
Fenster:cd ..

This will change your directory to the folder that your current folder was in. So if you're in Desktop > Git folder, using cd.. will change your directory to Desktop.

MKDIR

MKDIR steht für Make Directory. Dies wird verwendet, um neue Ordner in Ihrem aktuellen Pfad zu erstellen. Während wir uns noch im Desktop befinden, erstellen wir einen neuen Ordner namens git. Stellen Sie sicher, dass es sich um ein kleines Gehäuse handelt, damit es einfacher ist, zu tippen.

macOS:mkdir git
Fenster:mkdir git

Now in your desktop, a new folder called git has appeared!

CLEAR / CLS

Das Terminalfenster kann sehr schnell super chaotisch werden! So löschen Sie unser aktuelles Fenster:

macOS:clear
Fenster:cls

Dadurch wird Text aus dem Terminalfenster gelöscht und mit einem frisch aussehenden Bildschirm begonnen.
