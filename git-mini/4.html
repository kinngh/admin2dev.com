Markdown

The Markdown syntax is how we format text. A markdown file ends with the .md or .markdown extension. Before we get into it, let's make a new repository on GitHub.

Después de iniciar sesión, haga clic en el botón verde "Nuevo" a la izquierda para crear un nuevo repositorio.

Ahora rellene el nombre y los detalles sobre su nuevo repositorio. Asegúrese de que "Inicialice este repositorio con un README.MD" esté marcado.¡Ahora serás recibido con tu nuevo repositorio! Haga clic en el archivo "README.MD"Ahora haga clic en el botón de ediciónSe le recibirá con una pantalla similar a esta donde puede ingresar su texto.También está la sección de vista previa que le muestra lo que se ha cambiado, eliminado y agregado en su archivo. Vamos a ir más sobre las diferencias en un capítulo posterior. ¡Por ahora, aprendamos rebajas!

Título

Headings range from 1 to 5, with 1 being the largest and 5 the smallest. To write a line of heading, we add # before the text.

Markdown Salida
# Heading 1

Rúbrica 1
## Heading 2

Rúbrica 2
### Heading 3

Rúbrica 3
#### Heading 4

Rúbrica 4
##### Heading 5
Rúbrica 5

Énfasis

Para enfatizar el texto, podemos usar negrita, cursiva o strike.

Markdown Salida
**bold** negrita
_italics_ cursiva
~strike~ huelga

Listas

Markdown le permite tener tres tipos de lista.

Listas ordenadas

Markdown Salida
1. Uno
2. Dos
3. Tres
  1. Uno
  2. Dos
  3. Tres

Listas desordenadas

Markdown Salida
- Uno
- Dos
- Tres
  • Uno
  • Dos
  • Tres

Sangrías y combinación

Las listas ordenadas y desordenadas se pueden combinar, junto con sangría con pestañas para crear listas anidadas.

Markdown Salida
1. Uno
2. Dos
- Nido 1
- Nido 2
  1. Uno
  2. Dos
    • Nido 1
    • Nido 2

Enlaces

Markdown permite hacer referencia a imágenes y archivos externos e internos (repo).

Imágenes

La sintaxis para agregar imágenes es![Image alt description](imageLink)

Interno

Markdown Salida
![Logo](folder/image/a2d_logo.png)

Externo

Markdown Salida
![Admin2Dev Logo](https://admin2dev.com/images/logo.png)

Archivos

La sintaxis para hacer referencia a los archivos es[Description Name](file Link)

Interno

Markdown Salida
[File Name](folder/file.txt) Nombre del archivo

Externo

Markdown Salida
[Batch Apex](https://github.com/Admin2Dev/apex-snippets/blob/master/batch/README.md) Ápice por lotes

Código

Hay dos maneras de marcar un bloque de texto como código en markdown.

Markdown Salida
`una sola línea` single line
```
multi
línea
código
```		 
multi
line
code

Mesa

Crear tablas en markdown puede ser confuso (e irritante) a veces, por lo que es mucho más fácil usar un servicio como Markdown Table Generator y pegar el código.
