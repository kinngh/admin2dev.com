Test de la semaine 2

Vous êtes parti jusqu'à la fin de la semaine 2 ! Ce test rapide est fait pour tester votre compréhension de base de tout ce que vous avez appris jusqu'à présent !

  • Vous pouvez regarder la syntaxe, les points bonus si vous ne le faites pas.
  • Idéalement, le test ne devrait pas prendre plus de 20 minutes à résoudre.
  • Tous les critères doivent être remplis.
  • La réponse est disponible sur Admin2Dev GitHub Repo.
  • La réponse et la ventilation complète du programme sont disponibles à la fin de cette page.

Le test

Question

Write a Trigger oppUpdate and a helper class must be named oppUpdateHelper with methods that:

  • Crée une nouvelle tâche chaque fois qu'une opportunité est insérée.
  • Crée une nouvelle tâche chaque fois qu'une opportunité est mise à jour, à moins qu'elle ne soit « Fermé perdu » ou « Fermé gagné ». Le sujet de la tâche doit correspondre à son stade actuel.
  • Crée une nouvelle tâche si l'opportunité passe de « Analyse des besoins » à « Fermé perdu ».

Démarrez une minuterie et notez combien de temps vous prenez pour résoudre. Idéalement, cela devrait prendre environ 30 minutes, mais ne vous inquiétez pas si vous passez par-dessus bord.

Réponse

Création d'une tâche lorsqu'une opportunité est insérée

Déclencheur

trigger oppUpdater on Opportunity (after insert) {

    if(Trigger.isafter){
        if (Trigger.isInsert){
            oppUpdaterHelper.oppInsertTask(trigger.new);

        }
    }
}

Aide de classe

public class oppUpdaterHelper{

    public static void oppInsertTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            Task newTask = new Task();
            newTask.WhatId = iterator.id;
            newTask.Subject = 'New task';
            finalList.add(newTask);
        }
        insert finalList;
    }

}

Créer une nouvelle tâche à moins que « Fermé gagné » ou « Fermé perdu »

Déclencheur

trigger oppUpdater on Opportunity (after insert) {

    if(Trigger.isafter){
        if (Trigger.isInsert){
            oppUpdaterHelper.oppInsertTask(trigger.new);
            oppUpdaterHelper.oppClosedTask(trigger.new);
        }
    }
}

Aide de classe

public class oppUpdaterHelper{

    public static void oppInsertTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            Task newTask = new Task();
            newTask.WhatId = iterator.id;
            newTask.Subject = 'New task';
            finalList.add(newTask);
        }
        insert finalList;
    }

    public static void oppClosedTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            if (iterator.StageName == 'Closed Won' || iterator.StageName == 'Closed Lost'){
                //Do nothing. Optionally, use != operator
            }
            else{
                Task newTask = new Task();
                newTask.WhatId = iterator.id;
                newTask.Subject = iterator.StageName;
                finalList.add(newTask);
            }
        }
        insert finalList;
    }

}

Crée une nouvelle tâche si l'opportunité passe de « Analyse des besoins » à « Fermé perdu ».

Déclencheur

trigger oppUpdater on Opportunity (after insert, after update) {

    if(Trigger.isafter){
        if (Trigger.isInsert){
            oppUpdaterHelper.oppInsertTask(trigger.new);
            oppUpdaterHelper.oppClosedTask(trigger.new);
        }

        if (Trigger.isUpdate){
            oppUpdaterHelper.oppChangeTask(trigger.oldMap, trigger.new);
        }
    }
}

Aide de classe

public class oppUpdaterHelper{

    public static void oppInsertTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            Task newTask = new Task();
            newTask.WhatId = iterator.id;
            newTask.Subject = 'New task';
            finalList.add(newTask);
        }
        insert finalList;
    }

    public static void oppClosedTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            if (iterator.StageName == 'Closed Won' || iterator.StageName == 'Closed Lost'){
                //Do nothing. Optionally, use != operator
            }
            else{
                Task newTask = new Task();
                newTask.WhatId = iterator.id;
                newTask.Subject = iterator.StageName;
                finalList.add(newTask);
            }
        }
        insert finalList;
    }

    public static void oppChangeTask(Map<ID, Opportunity> oldList, List<Opportunity> newList){


        for(Opportunity iterator: newList){

            Opportunity oldOpp = oldList.get(iterator.id);

            Boolean oldValue = oldOpp.stageName.equals('Needs Analysis');
            Boolean newValue = iterator.StageName.equals('Closed Lost');

            if(oldValue && newValue){
                Task newTask = new Task();
                newTask.WhatId = iterator.id;
                newTask.Subject = 'Closed Lost';
                insert newTask;
            }
        }
    }

}
