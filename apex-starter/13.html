Test in Woche 2

Sie haben es bis Ende der zweiten Woche geschafft! Dieser Schnelltest wurde durchgeführt, um Ihr grundlegendes Verständnis von allem, was Sie bisher gelernt haben, zu testen!

  • Sie können sich die Syntax ansehen, Bonuspunkte, wenn Sie es nicht tun.
  • Die Lösung des Tests sollte idealerweise nicht länger als 20 Minuten dauern.
  • Alle Kriterien müssen erfüllt sein.
  • Die Antwort ist unter Admin2Dev GitHub Repo verfügbar.
  • Die Antwort und die vollständige Aufschlüsselung des Programms finden Sie am Ende dieser Seite.

Der Test

Frage

Write a Trigger oppUpdate and a helper class must be named oppUpdateHelper with methods that:

  • Erstellt eine neue Aufgabe, wenn eine Verkaufschance eingefügt wird.
  • Erstellt eine neue Aufgabe, wenn eine Verkaufschance aktualisiert wird, es sei denn, sie ist "Verloren geschlossen" oder "Geschlossener Won". Der Aufgabenbetreff sollte mit seiner aktuellen Phase übereinstimmen.
  • Erstellt eine neue Aufgabe, wenn sich die Verkaufschance von "Needs Analysis" in "Geschlossen verloren" ändert.

Starten Sie einen Timer und notieren Sie sich, wie lange Sie brauchen, um ihn zu lösen. Idealerweise sollte es ~30 Minuten dauern, aber keine Sorge, wenn Sie über Bord gehen.

Antwort

Erstellen einer Aufgabe beim Einfügen einer Absatzchance

Auslöser

trigger oppUpdater on Opportunity (after insert) {

    if(Trigger.isafter){
        if (Trigger.isInsert){
            oppUpdaterHelper.oppInsertTask(trigger.new);

        }
    }
}

Klassenhelfer

public class oppUpdaterHelper{

    public static void oppInsertTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            Task newTask = new Task();
            newTask.WhatId = iterator.id;
            newTask.Subject = 'New task';
            finalList.add(newTask);
        }
        insert finalList;
    }

}

Erstellen Sie eine neue Aufgabe, es sei denn, "Geschlossen gewonnen" oder "Geschlossen verloren"

Auslöser

trigger oppUpdater on Opportunity (after insert) {

    if(Trigger.isafter){
        if (Trigger.isInsert){
            oppUpdaterHelper.oppInsertTask(trigger.new);
            oppUpdaterHelper.oppClosedTask(trigger.new);
        }
    }
}

Klassenhelfer

public class oppUpdaterHelper{

    public static void oppInsertTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            Task newTask = new Task();
            newTask.WhatId = iterator.id;
            newTask.Subject = 'New task';
            finalList.add(newTask);
        }
        insert finalList;
    }

    public static void oppClosedTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            if (iterator.StageName == 'Closed Won' || iterator.StageName == 'Closed Lost'){
                //Do nothing. Optionally, use != operator
            }
            else{
                Task newTask = new Task();
                newTask.WhatId = iterator.id;
                newTask.Subject = iterator.StageName;
                finalList.add(newTask);
            }
        }
        insert finalList;
    }

}

Erstellt eine neue Aufgabe, wenn sich die Verkaufschance von "Needs Analysis" in "Geschlossen verloren" ändert.

Auslöser

trigger oppUpdater on Opportunity (after insert, after update) {

    if(Trigger.isafter){
        if (Trigger.isInsert){
            oppUpdaterHelper.oppInsertTask(trigger.new);
            oppUpdaterHelper.oppClosedTask(trigger.new);
        }

        if (Trigger.isUpdate){
            oppUpdaterHelper.oppChangeTask(trigger.oldMap, trigger.new);
        }
    }
}

Klassenhelfer

public class oppUpdaterHelper{

    public static void oppInsertTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            Task newTask = new Task();
            newTask.WhatId = iterator.id;
            newTask.Subject = 'New task';
            finalList.add(newTask);
        }
        insert finalList;
    }

    public static void oppClosedTask(List<Opportunity> oppList){

        List<Task> finalList = new List<Task>();

        for (Opportunity iterator: oppList){
            if (iterator.StageName == 'Closed Won' || iterator.StageName == 'Closed Lost'){
                //Do nothing. Optionally, use != operator
            }
            else{
                Task newTask = new Task();
                newTask.WhatId = iterator.id;
                newTask.Subject = iterator.StageName;
                finalList.add(newTask);
            }
        }
        insert finalList;
    }

    public static void oppChangeTask(Map<ID, Opportunity> oldList, List<Opportunity> newList){


        for(Opportunity iterator: newList){

            Opportunity oldOpp = oldList.get(iterator.id);

            Boolean oldValue = oldOpp.stageName.equals('Needs Analysis');
            Boolean newValue = iterator.StageName.equals('Closed Lost');

            if(oldValue && newValue){
                Task newTask = new Task();
                newTask.WhatId = iterator.id;
                newTask.Subject = 'Closed Lost';
                insert newTask;
            }
        }
    }

}
