Operaciones condicionales, más operaciones aritméticas y bucles

Tipo de datos booleanos

El tipo de datos booleano solo puede contener valores que son verdaderos o falsos. Aquí hay una sintaxis rápida:

Boolean variableName = value;
  • Booleanes la declaración de tipo de datos.
  • variableNamees el nombre asignado a la variable.
  • =es asignar el valor a la variable.
  • value can only be true or false and doesn't use any '.

Aquí hay ejemplos rápidos de cómo declarar variables booleanas:

Boolean a = true;
Boolean b = false;

Operadores de comparación

Utilizamos operadores de comparación para comparar los valores de dos variables similares. Los operadores de comparación son una parte integral de la programación porque ayuda a ejecutar ciertos bloques de código cuando se cumplen condiciones específicas. Aquí están los operadores de comparación:

  • == is different from = operator. A single = assigns the value on the right to the variable on the left but the == operator checks if the value on the left IS EQUAL to the value on the right.
  • !=comprueba si el valor de la derecha NO ES IGUAL a la variable de la izquierda. Esto devuelve true si los valores son diferentes, y false si los valores son los mismos.
  • >comprueba si el valor de la izquierda es mayor que la variable de la izquierda.
  • <comprueba si el valor de la derecha es mayor que la variable de la izquierda.
  • <=comprueba si el valor de la derecha es mayor o igual a la variable de la izquierda.
  • >=comprueba si el valor de la derecha es menor o igual a la variable de la izquierda.
  • ||se llama el operador OR. Toma dos valores en cada lado y comprueba que devuelve true si cualquiera de los valores es true, o ambos valores son true y devuelve false si ambos valores son false.
  • &&se llama el operador AND. Toma dos valores a cada lado y devuelve true si ambos valores son true, y false si cualquiera de los valores es false, o ambos valores son false.

Condicionales

Un condicional muy utilizado es la declaración if-else. Es muy sencillo y sencillo de usar. Aquí está la sintaxis:

if(condition){
  //run if code
} else {
  //run else code
}
  • ifdeclara el inicio de la condición.
  • (condition)tiene una combinación de operadores de comparación y valores/variables. Si la condición se evalúa como true, se ejecuta el bloque de código if y si es false, todo el bloque de código se ignora y se mueve a otra. Es una buena práctica mantener siempre la condición entre corchetes.
  • //run if code theEl símbolo // se utiliza para comentar en código. Cualquier cosa hacia el lado derecho de la // en la línea no es ejecutada por la computadora.
  • elsese ejecuta si la primera condición se evalúa como falsa.
  • { }los corchetes se utilizan para denotar bloques de código.
  • Notice there is no ; to mark the end of the if-else condition. The values inside the code block, however, require ;.

Aquí hay un ejemplo de una simple sentencia if-else:

Integer a = 4;
if (a==4){
  System.debug('A is 4!');
} else {
  System.debug('A is not 4!');
}

¡Cambie el valor de a a cualquier otro valor y vea los resultados! La instrucción if-else se puede encadenar para formar instrucciones if-else más grandes:

Integer a = 4;
if (a==4){
    System.debug('A is 4!');
} else if (a==5) {
    System.debug('A is 5!');
} else {
    System.debug('A is not 4 or 5!');
}

Una instrucción if-else también se puede usar sin la instrucción else.

Integer a = 4;
if (a==4){
    System.debug('A is 4!');
  }
if (a==5) {
    System.debug('A is 5!');
}

Las sentencias If-else se pueden poner una dentro de la otra. Esto se llama una instrucción if-else anidada.

Integer a = 4;
if (a==4 || a == 5){
  if (a==4){
    System.debug('A is 4!');
  } else {
    System.debug('A is 5!');
  }
} else {
  System.debug('A is not 4 or 5');
}

El código utiliza una instrucción OR para comprobar el valor de a. La línea 2 se evalúa como verdadera si a es 4 o 5.

Encadenar instrucciones if-else puede complicarse mucho, y para lidiar con eso, usamos instrucciones switch. Aquí está la sintaxis:

switch on variableName{
    when value1{
      //run code block 1
    }
    when value2 {
      //run code block 2
    }
    when else {
      //run code block 3
    }
}
  • switch onse utiliza para declarar que estamos haciendo una declaración switch
  • variableNameEsta es la variable en la que estamos ejecutando la instrucción switch.
  • when value1 if the value of variableName is value1, the code block 1 is run
  • when elseeste bloque de código se ejecutará si el valor de variableName no es igual a ningún valor.

Aquí hay un ejemplo

Integer a = 10;
switch on a{
  when 10{
    System.debug('A is 10');
  }
  when 20{
    a++;
    System.debug('A is ' + a);
  }
  when else {
    a *= 40;
    System.debug('A is ' + a);
  }
}

Resumen

  • Los booleanos solo pueden tomar true o false como sus valores.
  • Los operadores de comparación ayudan a comparar valores.
  • Las instrucciones if-else se pueden anidar y solo se pueden ejecutar como una instrucción if.
  • Las instrucciones Switch son una buena manera de refactorizar las sentencias if-else cuando necesitan anidarse mucho.
