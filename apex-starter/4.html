Opérations conditionnelles, opérations plus arithmétiques et boucles

Type de données booléennes

Le type de données booléennes ne peut contenir que des valeurs vraies ou fausses. Voici une syntaxe rapide :

Boolean variableName = value;
  • Booleanest la déclaration de type de données.
  • variableNameest le nom attribué à la variable.
  • =est d'attribuer la valeur à la variable.
  • value can only be true or false and doesn't use any '.

Voici des exemples rapides de la façon de déclarer les variables booléennes :

Boolean a = true;
Boolean b = false;

Opérateurs de comparaison

Nous utilisons des opérateurs de comparaison pour comparer les valeurs de deux variables similaires. Les opérateurs de comparaison font partie intégrante de la programmation car ils aident à exécuter certains blocs de code lorsque des conditions spécifiques sont remplies. Voici les opérateurs de comparaison :

  • == is different from = operator. A single = assigns the value on the right to the variable on the left but the == operator checks if the value on the left IS EQUAL to the value on the right.
  • !=vérifie si la valeur de droite N'EST PAS ÉGALE par rapport à la variable de gauche. Cela renvoie true si les valeurs sont différentes, et false si les valeurs sont les mêmes.
  • >vérifie si la valeur à gauche est supérieure à la variable à gauche.
  • <vérifie si la valeur de la droite est supérieure à la variable de gauche.
  • <=vérifie si la valeur de la droite est supérieure ou égale à la variable de gauche.
  • >=vérifie si la valeur de la droite est inférieure ou égale à la variable de gauche.
  • ||est appelé l'opérateur OU. Il prend deux valeurs de chaque côté et vérifie que true renvoie si l'une ou l'autre valeur est vraie, ou si les deux valeurs sont vraies et renvoie false si les deux valeurs sont fausses.
  • &&s'appelle l'opérateur AND. Il prend deux valeurs de chaque côté et renvoie true si les deux valeurs sont vraies, et false si l'une ou l'autre valeur est fausse, ou les deux valeurs sont fausses.

Conditionnels

Un conditionnel très largement utilisé est l'instruction if-else. C'est très simple et simple à utiliser. Voici la syntaxe :

if(condition){
  //run if code
} else {
  //run else code
}
  • ifdéclare le début de la condition.
  • (condition)contient une combinaison d'opérateurs de comparaison et de valeurs/variables. Si la condition est évaluée à true, le bloc de code if est exécuté et s'il est false, le bloc de code entier est ignoré et passe à else. C'est une bonne pratique de toujours garder l'état entre parenthèses.
  • //run if code the// symbole est utilisé pour commenter dans le code. Tout ce qui se trouve vers le côté droit de la // dans la ligne n'est pas exécuté par l'ordinateur.
  • elseest exécuté si la première condition est évaluée à false.
  • { }les crochets sont utilisés pour désigner les blocs de code.
  • Notice there is no ; to mark the end of the if-else condition. The values inside the code block, however, require ;.

Voici un exemple d'une simple instruction if-else :

Integer a = 4;
if (a==4){
  System.debug('A is 4!');
} else {
  System.debug('A is not 4!');
}

Changez la valeur de a par n'importe quelle autre valeur et voyez les résultats ! L'instruction if-else peut être enchaînée pour former des instructions if-else plus grandes :

Integer a = 4;
if (a==4){
    System.debug('A is 4!');
} else if (a==5) {
    System.debug('A is 5!');
} else {
    System.debug('A is not 4 or 5!');
}

Une instruction if-else peut également être utilisée sans l'instruction else.

Integer a = 4;
if (a==4){
    System.debug('A is 4!');
  }
if (a==5) {
    System.debug('A is 5!');
}

Les instructions If-else peuvent être placées les unes dans les autres. C'est ce qu'on appelle une instruction imbriquée if-else.

Integer a = 4;
if (a==4 || a == 5){
  if (a==4){
    System.debug('A is 4!');
  } else {
    System.debug('A is 5!');
  }
} else {
  System.debug('A is not 4 or 5');
}

Le code utilise une instruction OU pour vérifier la valeur de a. La ligne 2 est évaluée à true si a vaut 4 ou 5.

Enchaîner les instructions if-else peut devenir vraiment compliqué, et pour y faire face, nous utilisons des instructions switch. Voici la syntaxe :

switch on variableName{
    when value1{
      //run code block 1
    }
    when value2 {
      //run code block 2
    }
    when else {
      //run code block 3
    }
}
  • switch onest utilisé pour déclarer que nous faisons une instruction switch
  • variableNameC'est la variable sur laquelle nous exécutons l'instruction switch.
  • when value1 if the value of variableName is value1, the code block 1 is run
  • when elsece bloc de code s'exécutera si la valeur de variableName n'est égale à aucune valeur.

Voici un exemple

Integer a = 10;
switch on a{
  when 10{
    System.debug('A is 10');
  }
  when 20{
    a++;
    System.debug('A is ' + a);
  }
  when else {
    a *= 40;
    System.debug('A is ' + a);
  }
}

Résumé

  • Les booléens ne peuvent prendre que true ou false comme valeurs.
  • Les opérateurs de comparaison aident à comparer les valeurs.
  • Les instructions if-else peuvent être imbriquées et ne peuvent être exécutées que comme une instruction if.
  • Les instructions Switch sont un bon moyen de refactoriser les instructions if-else lorsqu'elles ont besoin d'être beaucoup imbriquées.
