Bedingte Operationen, arithmetischere Operationen und Schleifen

Boolescher Datentyp

Der boolesche Datentyp kann nur Werte enthalten, die wahr oder falsch sind. Hier ist eine schnelle Syntax:

Boolean variableName = value;
  • BooleanIst die Datentypdeklaration.
  • variableNameIst der Name, der der Variablen zugewiesen ist.
  • =Ist es, den Wert der Variablen zuzuweisen.
  • value can only be true or false and doesn't use any '.

Hier sind kurze Beispiele für die Deklaration boolescher Variablen:

Boolean a = true;
Boolean b = false;

Vergleichsoperatoren

Wir verwenden Vergleichsoperatoren, um die Werte von zwei ähnlichen Variablen zu vergleichen. Vergleichsoperatoren sind ein integraler Bestandteil der Programmierung, da sie bei der Ausführung bestimmter Codeblöcke hilft, wenn bestimmte Bedingungen erfüllt sind. Hier sind Vergleichsoperatoren:

  • == is different from = operator. A single = assigns the value on the right to the variable on the left but the == operator checks if the value on the left IS EQUAL to the value on the right.
  • !=Prüfer prüft, ob der Wert auf der rechten Seite NICHT WERTIG zu der Variablen auf der linken Seite ist. Dies gibt TRUE zurück, wenn die Werte unterschiedlich sind, und FALSE, wenn die Werte identisch sind.
  • >Prüfer prüft, ob der Wert auf der linken Seite größer als die Variable auf der linken Seite ist.
  • <Prüfer prüft, ob der Wert der rechten Seite größer als die Variable auf der linken Seite ist.
  • <=Prüfer prüft, ob der Wert der rechten Seite entweder größer oder gleich der Variablen auf der linken Seite ist.
  • >=Prüfer prüft, ob der Wert der rechten Seite entweder kleiner oder gleich der Variablen auf der linken Seite ist.
  • ||Wird als ODER-Operator bezeichnet. Es nimmt zwei Werte auf beiden Seiten und prüft TRUE, ob einer der Werte wahr ist oder beide Werte wahr sind, und gibt FALSE zurück, wenn beide Werte falsch sind.
  • &&Wird als AND-Operator bezeichnet. Es nimmt zwei Werte auf beiden Seiten und gibt true zurück, wenn beide Werte wahr sind, und false, wenn einer der Werte falsch oder beide Werte falsch ist.

Bedingte Bedingungen

A very widely used conditional is the if-else statement. It's very straight forward and simple to use. here is the syntax: 

if(condition){
  //run if code
} else {
  //run else code
}
  • ifDeklariert den Beginn der Bedingung.
  • (condition)Enthält eine Kombination aus Vergleichsoperatoren und Werten/Variablen. Wenn die Bedingung als wahr ausgewertet wird, wird der if-Codeblock ausgeführt und wenn er falsch ist, wird der gesamte Codeblock ignoriert und zu else verschoben. Es ist eine gute Praxis, den Zustand immer in Klammern zu halten.
  • //run if code the// Symbol wird verwendet, um im Code zu kommentieren. Alles auf der rechten Seite des // in der Zeile wird nicht vom Computer ausgeführt.
  • elseWird ausgeführt, wenn die erste Bedingung als falsch ausgewertet wird.
  • { }Klammern werden verwendet, um Codeblöcke zu bezeichnen.
  • Notice there is no ; to mark the end of the if-else condition. The values inside the code block, however, require ;.

Hier ist ein Beispiel für eine einfache if-else-Anweisung:

Integer a = 4;
if (a==4){
  System.debug('A is 4!');
} else {
  System.debug('A is not 4!');
}

Ändern Sie den Wert von a in einen anderen Wert und sehen Sie sich die Ergebnisse an! Die if-else-Anweisung kann verkettet werden, um größere if-else-Anweisungen zu bilden:

Integer a = 4;
if (a==4){
    System.debug('A is 4!');
} else if (a==5) {
    System.debug('A is 5!');
} else {
    System.debug('A is not 4 or 5!');
}

Eine if-else-Anweisung kann auch ohne die else-Anweisung verwendet werden.

Integer a = 4;
if (a==4){
    System.debug('A is 4!');
  }
if (a==5) {
    System.debug('A is 5!');
}

If-else-Anweisungen können ineinander eingefügt werden. Dies wird als verschachtelte if-else-Anweisung bezeichnet.

Integer a = 4;
if (a==4 || a == 5){
  if (a==4){
    System.debug('A is 4!');
  } else {
    System.debug('A is 5!');
  }
} else {
  System.debug('A is not 4 or 5');
}

Der Code verwendet eine OR-Anweisung, um den Wert einer zu überprüfen. Zeile 2 wird als wahr ausgewertet, wenn a entweder 4 oder 5 ist.

Chaining if-else statements can get really complicated, and to deal with that, we use switch statements. Here is the syntax:

switch on variableName{
    when value1{
      //run code block 1
    }
    when value2 {
      //run code block 2
    }
    when else {
      //run code block 3
    }
}
  • switch onWird verwendet, um zu deklarieren, dass wir eine switch-Anweisung abgeben
  • variableNameDies ist die Variable, auf der wir die switch-Anweisung ausführen.
  • when value1 if the value of variableName is value1, the code block 1 is run
  • when elseDieser Codeblock wird ausgeführt, wenn der Wert von variableName keinem Wert entspricht.

Hier ist ein Beispiel

Integer a = 10;
switch on a{
  when 10{
    System.debug('A is 10');
  }
  when 20{
    a++;
    System.debug('A is ' + a);
  }
  when else {
    a *= 40;
    System.debug('A is ' + a);
  }
}

Zusammenfassung

  • Boolesche Werte können nur wahr oder falsch als ihre Werte annehmen.
  • Vergleichsoperatoren helfen beim Vergleich von Werten.
  • if-else-Anweisungen können verschachtelt und nur als if-Anweisung ausgeführt werden.
  • Switch-Anweisungen sind eine gute Möglichkeit, if-else-Anweisungen umzugestalten, wenn sie viel verschachtelt werden müssen.
