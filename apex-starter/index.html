Apex-Startkurs

Willkommen beim Starter Apex Course! Dies ist eine kostenlose Sammlung von Tutorials, Videos, Tests und mehr, die Ihnen helfen sollen, Apex effektiv zu lernen.

Dieser Kurs richtet sich an Salesforce-Administratoren, die zur Entwicklerrolle wechseln oder ihre Fähigkeiten über deklarative Tools hinaus erweitern möchten.

Ich habe versucht, diesen Kurs so zu beschleunigen, dass er jeden Tag weniger als eine Stunde dauert, so dass Sie selbst wenn Sie stecken bleiben, eine Lösung finden können und schneller zu lernen, werden keine besseren Ergebnisse erzielen. Nehmen Sie sich Zeit!

Sie können mich jederzeit @kinngh twittern oder unserem Discord-Server beitreten, um Hilfe zu erhalten.
