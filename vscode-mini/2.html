Entdecker

Explorer enthält eine Liste aller Verzeichnisse und Dateien in Ihrem Projekt. Während wir dies in den kommenden Kapiteln lernen werden, hier ist ein Screenshot davon, wie Ihr Bildschirm aussehen wird, wenn Sie ein Projekt geöffnet haben, mit einer testClass-Vorlage.

  • Open Editors shows all currently open files. In the screenshot, we have our actual testClass.cls file that contains the testClass class, and testClass.cls-meta.xml that contains information about the test class, that VSCode auto generates. This is a SOAP API that is used to transfer our code to the org we are working in, and is handled automatically.
  • Test Org ist der Name, den ich meinem gerade erstellten Projekt gegeben habe. Wir gehen dies durch, wenn wir über die Verbindung von VSCode mit unserer Organisation diskutieren, und werden in den kommenden Kapiteln klar sein. Wissen Sie nur, dass jedes einzelne Projekt, das Sie erstellen, ein neuer Unterabschnitt mit dem Namen hinzugefügt wird.
  • Outline enthält alle Klassen und Objekte, die wir erstellt haben.
  • Die Zeitleiste enthält eine Historie aller unserer Git-Commits und unseres Codes.
  • Ausführen von Aufgaben listet alle derzeit ausgeführten Aufgaben auf.

Another really great feature in VSCode is the mini map available on the right side. This is handy when you have a file that contains a lot of code and having a mini map allows you to navigate more quickly. Let's populate the testClass with random methods and see it in action:

Suche

Die Suche ist genau das, was Sie erwarten, eine Suchleiste auf der linken Seite, geben Sie Ihre Abfrage ein, drücken Sie die Eingabetaste und Sie haben Ergebnisse zur Verfügung.

Quellenkontrolle

Die Quellcodeverwaltung dient der Versionierung unserer Arbeit und unserer Dateisysteme. Es verwendet Git und hat integrierten Zugriff, um direkt auf GitHub zu veröffentlichen. Wir gehen mehr darüber in einem separaten Mini-Kurs Git and Collaboration Mini-Kurs durch, der erklärt, was Git ist, Einrichtung und wie man mit anderen Benutzern über Git zusammenarbeitet. Im Moment ignorieren wir dies wettbewerbsfähig, weil es für diesen Minikurs nicht in den Geltungsbereich fällt.

Einstellungen

All your settings live here. This is also where you can select theme for VSCode. I am using the Dark+ (default dark) theme. This only changes your color scheme, so it's safe to play around with. You can get more themes from extensions panel by searching for category:themes or heading over to my recommended VSCode Theme website, VSCodeThemes. (Not affiliated in any way.), or you can search the internet for more VSCode themes if you like.
