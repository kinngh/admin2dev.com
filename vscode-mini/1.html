Configuration de VSCode pour le développement

Prenez ce scénario. Vous venez de développer cet intérêt pour le café et maintenant vous voulez passer de votre poudre de café instantané et de votre lait à quelque chose de plus sérieux. Vous allez en ligne, recherchez des brasseries de café locales et planifiez un programme de formation au café avec elles. Une semaine plus tard, vous avez un processus sophistiqué de fabrication du café qui vous est unique. Votre café est préparé dans une chambre sous vide afin que vous obteniez chaque goutte de café de vos grains avant de l'incorporer à l'expresso parfait.

Votre ensemble d'outils compte. Bien sûr, la console de développement est un excellent mélange de café instantané, vous avez besoin de quelque chose de plus puissant parce que la console de développement ne peut que vous mener jusqu'à présent. Lorsque vous commencez à grandir en tant que développeur, vous avez envie de l'endroit idéal pour construire et déboguer vos programmes et ce mini-cours vous présente votre chambre d'aspirateur pour infuser votre café.

Visual Studio Code

Visual Studio Code, ou VSCode, est un IDE construit et publié par Microsoft et est disponible gratuitement. La grande chose à propos de VSCode est qu'il est vraiment personnalisable et que l'industrie adopte rapidement VSCode en tant qu'IDE standard en dehors de la console de développement. Maintenant, configurons VSCode.

Tout d'abord, nous devons télécharger JDK ou Java Development Kit sur le site Web d'Oracle disponible ici. Même si vous avez JDK installé, installez-le à nouveau. C'est pour vous assurer que vous êtes toujours sur la dernière version du logiciel et vraiment, c'est tout simplement plus facile de cette façon. J'ai vu des gens rencontrer des problèmes et ne pas être en mesure de comprendre les choses pendant des semaines à la fois, seulement pour réinitialiser complètement leur système et démarrer leurs machines à partir de zéro.

Téléchargez maintenant VSCode sur le site Web de leur Visual Studio pour votre plate-forme et installez-le et ouvrez-le.

The first thing to do after you install and open it, is to go in Settings and in Commonly Used section, change Auto Save to any option you prefer. I have kept it on afterDelay set to 10000 so I don't have to worry about saving the file I am working on. The delay is in ms where 1 second = 1000 ms, so autosaving my file every 10 seconds works well.

Now open the extensions section and search for Salesforce Extension Pack and install it.

Pendant l'installation, nous avons besoin d'outils Salesforce CLI sur notre machine. Rendez-vous sur le site Web Salesforce CLI et installez-le pour votre plate-forme.

Une fois terminé, il affichera une fenêtre contextuelle en bas de votre écran.

Restart your computer. I've noticed this with multiple instances that developers don't restart their computer for days and run into errors, and yes, I've helped debug people from our Discord community with this, where the solution was simply, restart your computer. After your restart, open VSCode and press Command + Shift + P for macOS users or Control + Shift + P for Windows users and type SFDX. If you see commands popping up, you have correctly configured everything!
