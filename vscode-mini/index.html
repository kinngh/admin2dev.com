VSCode Mini Course (Beta)

Welcome to VSCode Mini Course! This is a free collection of text and video tutorials designed to help you setup VSCode to write, run and execute Apex Code

This is a quick mini course for Administrators Developers fresh out of the Apex Starter Course

This course goes over quickly setting up VSCode and getting familiar with it's UI, and get to writing Apex code and deploying to our developer org!

You can always tweet me @theharshdeep or join our Discord Server for help.

Chapter 7 in this course is a live chapter and will be updated based on community feedback.

Prerequisites

  • Apex Starter Course

Tools

A list of tools that you will be using through out the course. There are chapters for setting up all the tools when we need them.

  • Salesforce CLI Tool

  • Java Development Kit (JDK)
