Verbindung zur Salesforce-Organisation herstellen

Dies sind Schritte, um Ihre VSCode-Instanz mit Ihrer Organisation zu verbinden. Es gibt hier nichts zu verstehen, nur Standardschritte, die zu befolgen sind.

Open VSCode and press Command + Shift + P for macOS users or Control + Shift + P for Windows users and search for SFDX: Create a new project with manifest.

Nach ein paar Sekunden werden Sie mit drei Optionen begrüßt. WählenEmpty

Anschließend werden Sie aufgefordert, Ihr Projekt zu benennen. Wir werden uns mit unserer Entwicklerorganisation verbinden, also nennen wir esMy Developer Org

Next, it will open your file explorer and ask you where you want to save your project folder. I usually make a new folder in my Desktop called vscode folder and save it inside the folder. Your project folder will include all the files that you will be writing and uploading, so make sure you save it in a place you can easily access and remember.

Sobald dies erledigt ist, werden Sie mit einem ähnlichen Bildschirm begrüßt:

Press Command + Shift + P for macOS users or Control + Shift + P for Windows users and search for SFDX: Authorize an org.

Benutzerdefiniert auswählen

Now we need to grab the URL of our developer org. For this, sign in to your developer org, open Setup and Switch to Salesforce Classic. Once there, copy the first part of the url. For my case, my full url is https://ap4.salesforce.com/home/home.jsp?source=lex but I need only the https://ap4.salesforce.com. This will be different for everyone, but the format stays the same. It's always https://xxx.salesforce.com.

Kopieren Sie diese URL in den Abschnitt "Benutzerdefinierte Anmelde-URL" von VSCode und drücken Sie die Eingabetaste.

Now we need to name this org. I am naming mine DeveloperOrg and press enter.

This will trigger a login screen in your default browser, login and authorize Salesforce CLI to access your org.

Klicken Sie nun in Ihrer unteren blauen Leiste aufNo default org set

This will trigger a popup at top, select the Developer Org - name@email.com we just authorized

Wenn alles gut läuft, sollte die untere Leiste aktualisiert werden, um etwa so auszusehen:

Ihr letzter Bildschirm sollte wie folgt aussehen:

  • You have your project name open in the left panel in explorer with a bunch of folders like .sfdx, config, etc.
  • In der unteren Leiste wird der Name der Organisation angezeigt, die Sie gerade verbunden haben.
