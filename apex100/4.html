    Boolean Data Type

    The boolean data type can only hold values that are true or false. Here is a quick syntax:

    Boolean variableName = value;
    • Boolean is the data type declaration.
    • variableName is the name assigned to the variable.
    • = is to assign the value to the variable.
    • value can only be true or false and doesn't use any '.

    Here are quick examples on how to declare boolean variables:

    Boolean a = true;
Boolean b = false;

    Conditionals

    A very widely used conditional is the if else statement. It's very straight forward and simple to use. here is the syntax: 

    if(condition){
  //run if code
} else {
  //run else code
}
    • if is the first condition.
    • (condition) if the condition evaluates to true, the statement is run. It's a good practice to always keep the condition within brackets.
    • //run if code the // symbol is used to comment in code. Anything towards the right side of the // in the line is not run by the computer.
    • else is run if the first condition evaluates to false.
    • { } brackets are used to denote code blocks.

    Here's an example to a simple if-else statement:

    Integer a = 4;
if (a==4){
  System.debug('A is 4!');
} else {
  System.debug('A is not 4!');
}

    Change the value of a to any other value and see the results! The if-else statement can be chained to form bigger if-else statements:

    Integer a = 4;
if (a==4){
    System.debug('A is 4!');
  } else if (a==5) {
    System.debug('A is 5!');
  } else {
    System.debug('A is not 4 or 5!');
}

    An if-else statement can also be used without the else statement.

    Integer a = 4;
if (a==4){
    System.debug('A is 4!');
  }
if (a==5) {
    System.debug('A is 5!');
}

    Chaining if-else statements can get really complicated, and to deal with that, we use switch statements. Here is the syntax:

    switch variableName{
case n:
  //run code if variableName == n;
  break;
case o:
  //run code if variableName == o;
  break;
default:
  //run code if variableName has any other value except n or o
}
    • switch is used to declare we are making a switch statement
    • variableName this variable should exist and we are running the switch statement on this variable.
    • case n: this is the code block we are running if the value of variableName is n
    • break; tells the computer to not run any code after break
    • default: this code block will run if the value of variableName isn't equal to any cases we have made.

    Here's an example

    Integer a = 10;
switch a{
  case 10:
    System.debug(a);
    break;
  case 20:
    a++;
    System.debug(a);
    break;
  default:
    a *= 40;
    System.debug(a);
}
    • If the value of a is 10, it will output the value of a in the log.
    • If the value of a is 20, it will add 1 to it and output it in the log.
    • If the value of a is NOT 10 or 20, it will multiply 40 to a and output it in the log.

    What happens if you don't write the break;? The code will continute to run until it hits a default or a break inside a different case. Try this block of code:

    Integer a = 10;
switch a{
  case 10:
    System.debug(a);
    break;
  case 20:
    a++;
    System.debug(a);
    //No brake here
  default:
    a *= 40;
    System.debug(a);
}

    What value(s) do you see in the log?
